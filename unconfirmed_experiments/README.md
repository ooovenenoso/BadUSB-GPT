# Unconfirmed experiments

This folder stores experimental scripts that have not yet been fully tested.

- `PentestGPT_4oMini.txt`: variant using the `gpt-4o-mini` model.
- `PentestGPT_4o.txt`: variant using the `gpt-4o` model.
- `PentestGPT_4Turbo.txt`: variant using the `gpt-4-turbo` model.

Before moving a payload to `confirmed_experiments/`, follow `CONFIRMATION_GUIDE.md` and record validation notes.
